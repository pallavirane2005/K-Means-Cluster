# 🎯 K-Means Customer Segmentation

An interactive machine learning web application that uses **K-Means Clustering** to segment retail customers based on their purchasing behavior. Built with a premium dark UI using Streamlit.

## ✨ Features
📊 Data Overview : Interactive exploration with statistics and distributions
🔍 Elbow Method : Automated WCSS analysis to find optimal K
🎯 K-Means Clustering : Visualize customer segments with centroids
📋 Results Export : Download segmented data as CSV
🎨 Premium UI : Dark theme with glassmorphism and neon effects

## 🛠️ Tech Stack

- **Python** — Core programming
- **Streamlit** — Interactive web UI
- **Scikit-learn** — K-Means clustering algorithm
- **Pandas** — Data manipulation
- **Matplotlib** — Data visualization
- **NumPy** — Numerical computations

## 📁 Dataset

**Mall Customers Dataset** from Kaggle

| Column | Description |
|--------|-------------|
| CustomerID | Unique customer identifier |
| Gender | Male / Female |
| Age | Customer age |
| Annual Income (k$) | Yearly income in thousands |
| Spending Score (1-100) | Spending behavior score |

🔗 [Dataset Source](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)
